Implementacion de un contrato ERC20

El usuario se deberá loguear con "admin" como usuario, "pass" como contraseña para modo administrador, en donde contará con un saldo de 100ETH y 100WETH

Otros usuarios son: "user1" "pass1" con 1ETH, y "user2" "pass2" sin fondos

En ambos casos, el usuario deberá elegir la red de Ethereum para ver datos con arrays y objetos

En proximas entregas se actualizaran las funciones, como enviar ETH de un usuario a otro, comprar NFTs, ver saldos, etc
